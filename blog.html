What is Document Object Model(DOM)

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to an element in the document.

How do you select an element from DOM

You can select an element from the DOM using various JavaScript methods. Here are some commonly used ones: 1. getElementById() – Selects an element by its id
2. getElementsByClassName() – Selects elements by class name (returns a collection)
3. getElementsByTagName() – Selects elements by tag name (returns a collection)
4. querySelector() – Selects the first element that matches a CSS selector

What is event delegation in the context of the DOM, and why it is useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element instead of adding event listeners to multiple child elements. The event then "bubbles up" from the target element to the parent, where it is handled.
Better Performance: Instead of adding event listeners to many elements, you use one listener on a common ancestor. Handles Dynamic Elements: Useful when new elements are added dynamically, as the parent can handle events for them. Less Memory Usage: Reduces the number of event listeners in the document.

How do you manupulate on element's attribute and styles using the DOM?

You can manipulate an element’s attributes (like src, href, alt, id, etc.) and styles dynamically using JavaScript.
